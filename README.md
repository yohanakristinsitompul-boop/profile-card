# 📖 Deskripsi Aplikasi

Profile Card merupakan aplikasi mobile sederhana yang memungkinkan pengguna membuat kartu profil digital. Aplikasi mendukung pengambilan foto menggunakan kamera maupun galeri, menyimpan data secara lokal menggunakan AsyncStorage, memperoleh koordinat GPS, serta melakukan build menjadi file APK menggunakan **Expo Application Services (EAS Build)** sehingga aplikasi dapat dijalankan tanpa Expo Go.

---

# ✨ Fitur Utama

✅ Input Data Profil

- Nama Lengkap
- NIM
- Program Studi
- Kelas

---

✅ Foto Profil

- 📷 Ambil foto menggunakan Kamera
- 🖼️ Pilih foto dari Galeri

---

✅ Lokasi

- 📍 Mengambil lokasi GPS
- Menampilkan Latitude & Longitude

---

✅ Penyimpanan Data

- 💾 Menyimpan data menggunakan AsyncStorage
- Data otomatis muncul kembali saat aplikasi dibuka

---

✅ Reset Data

- Menghapus seluruh data profil
- Menghapus foto
- Menghapus lokasi

---

✅ Build APK

- APK dibuat menggunakan **Expo EAS Build**
- Dapat diinstal langsung di Android tanpa Expo Go

# 🚀 Cara Menjalankan Project

### Clone Repository

```bash
git clone https://github.com/USERNAME/profile-card.git
```

Masuk ke folder project

```bash
cd profile-card
```

Install dependency

```bash
npm install
```

Menjalankan aplikasi

```bash
npx expo start
```

---

# 📱 Cara Install APK

1. Download file APK melalui link EAS Build.
2. Pindahkan APK ke perangkat Android (jika belum diunduh langsung di HP).
3. Buka file APK.
4. Izinkan instalasi dari sumber yang tidak dikenal jika diminta.
5. Tekan **Install**.
6. Setelah selesai, buka aplikasi **Profile Card**.
